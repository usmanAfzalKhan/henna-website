# Henna Artist Portfolio Website 🌿

A professionally designed, fully responsive portfolio website for a Henna artist. This site showcases the artist’s designs, services, client testimonials, and contact details in a sleek and modern layout, built with HTML, CSS, Bootstrap, and JavaScript.

## 🌐 Live Demo

👉 [View the website](https://mehndibysimra.netlify.app/)

## 💡 Features

- ✅ Typing animation on desktop; static text on mobile for performance
- ✅ Elegant dark translucent sections
- ✅ Animated background that pauses on hover over navbar/sections
- ✅ Fully responsive across devices (desktop, tablet, mobile)
- ✅ Image and video gallery with modal previews
- ✅ Realistic client reviews section
- ✅ Clean and user-friendly navigation

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **Bootstrap 5**
- **JavaScript (Vanilla)**
- **GitHub** for version control
- **Netlify** for deployment
